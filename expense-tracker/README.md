A modern, feature-rich expense tracking application that helps users monitor their spending, set budgets, and visualize financial data through beautiful, interactive charts.

![FinTrack Dashboard](https://via.placeholder.com/800x400?text=FinTrack+Dashboard+Screenshot)

## ✨ Features

### 👤 **Multi-User Support**
- Secure login/signup system
- Personalized user profiles
- Individual expense tracking per user

### 📊 **Interactive Dashboard**
- Real-time expense overview
- Spending by category (Pie Chart)
- Category breakdown (Bar Chart)
- Budget vs actual comparison
- Smart financial insights

### 💰 **Expense Management**
- Add/Edit/Delete expenses
- Categorize expenses (Rent, Food, Shopping, EMIs, etc.)
- Multiple payment methods (Cash, UPI, Card, Net Banking)
- Date-based tracking

### 🎯 **Budget Monitoring**
- Category-wise budget limits
- Visual budget progress bars
- Overspending alerts
- Monthly budget overview

### 📈 **Reports & Analytics**
- Monthly spending summary
- Top spending categories
- Savings rate calculation
- Transaction history with filters

### 🎨 **Modern UI/UX**
- Clean, intuitive interface
- Dark/Light theme support
- Responsive design (Mobile + Desktop)
- Smooth animations
- Real-time updates

## 🛠️ **Technologies Used**

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Emoji-based (no external dependencies)
- **State Management**: React Context API
- **Date Handling**: date-fns
- **Unique IDs**: uuid
